Apply for Short-Term Accommodation (CAS-2)

Use this service to apply for a Short-Term Accommodation placement on behalf of an applicant on Home Detention Curfew (HDC) licence.

You can save your progress and return to your application at any time.

For prison bail applicants, you need to complete the paper form.
{{ govukButton({ text: "Start now", href: "../prototypes/add-an-applicant/search-applicant--male.html", isStartButton: true }) }}

Before you start

You need:

  • consent from the applicant
  • the applicant's CRN
  • a Layer 3 OASys for the applicant, which has been updated in the last 6 months.
    • What to do if OASys is not up to date
    We recommend updating the applicant's OASys before starting so information can be imported to help you answer some sections. It's also important that information shared in the application is consistent with OASys.

    If you cannot update OASys, you can complete the sections manually.
