Apply for Short-Term Accommodation (CAS-2)

Use this service to apply for accommodation and support on behalf of someone on Home Detention Curfew (HDC) licence.

You can save your progress and return to your application at any time.

Prison bail applications must be submitted using the paper form.
{{ govukButton({ text: "Start now", href: "../prototypes/add-an-applicant/search-applicant--male.html", isStartButton: true }) }}

Before you start

You need:

  • consent from the applicant
  • the applicant's prison number (from NOMIS)
  • a Layer 3 OASys for the applicant, which has been updated in the last 6 months.
    • What to do if OASys is not up to date
    Information in the application should be up to date and consistent with OASys.

    If you can update OASys, do this before you start so risk information can be imported to save you time.

    If the applicant does not have an OASys or you cannot update it, you can complete the sections manually.
